# Changelog

Every rule change in this pipeline traces to a documented failure ("origin case" in SKILL.md). This file is the summary; the origin cases are the detail.

## 2026-07-17
- Added Step 10 (Release): publication is a step, not an exit. Readers become the pipeline's outermost check.
- Added the three-altitudes core principle: observable outcomes, downstream impact, and likely intent are graded separately. Mechanism-level analysis undercounts impact.
- Added the symmetric discipline clause to Step 6: steelman assertions must pass the Step 4 tests before they can move a ruling.
- Added the selection-effects test to Step 4: state who produced a pattern before the pattern is allowed as evidence about an actor.
- Post-mortems must credit human catches explicitly.

## 2026-07-14
- Added the over-correction check to Step 7 and the Counterexamples mechanism to the priors file: guarding against a prior can suppress evidence-supported claims. The bias check fires in both directions.
